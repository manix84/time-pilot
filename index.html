Welcome to my Time Pilot JS Remake.

Let me know what you think on Twitter or GitHub.

VERY Early Alpha

Be aware that this might break from time to time - If it does break, please check a my current Issues and raise a new one if you can't see yours listed. Thanks :)

Controlles:

  • Up: Up
  • Down: Down
  • Left: Left
  • Right: Right
  • Space-Bar: Fire/Shoot
  • P: Toggle Pause
  • F: Toggle Full-Screen (work in progress)
